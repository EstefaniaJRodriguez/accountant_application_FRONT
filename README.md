# Accountant App

Accountant aplication Front is a repository where you can find the frontend code for an app regarding to accountant services. This app was developed in REACT JS , using Express and Bootstrap React framework.

Currently, you can see the page on this domain. Please, dont use, it's the real and productive website.

- [https://www.genimpositivo.com/) uses [GEN Impositivo]([https://babeljs.io](https://www.genimpositivo.com/)/)

## Functionalities

- The customer choose the service want and pay for it in a locally payment application in Argentina. 
- The accountant receive the request and manage it.
- The customer can contact by form web to an accountant.
- The accountant receive in a panel all request and can change the request state and add observations.
